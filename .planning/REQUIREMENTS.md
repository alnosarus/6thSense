# Requirements

## v1 Requirements

### Positioning
- [ ] **POS-01**: User can understand the "surface + volumetric" sensing thesis in under 30 seconds on first view.
- [ ] **POS-02**: User can see at least three concrete technical proof points tied to current prototype status.

### Product Narrative
- [ ] **NAR-01**: User can understand the PNS layer (surface tactile sensing) from a dedicated section.
- [ ] **NAR-02**: User can understand the CNS layer (volumetric sensing) from a dedicated section.
- [ ] **NAR-03**: User can understand how both layers fuse into one robotic perception model.

### Conversion
- [ ] **CONV-01**: User can submit an email through a waitlist form CTA.
- [ ] **CONV-02**: User receives immediate inline feedback after form submit.

### Quality
- [ ] **QLT-01**: Landing page is responsive on mobile and desktop.
- [ ] **QLT-02**: Core content is accessible with semantic headings and labeled form fields.

## v2 Requirements (Deferred)

- [ ] **CONV-03**: Persist leads to CRM backend.
- [ ] **NAR-04**: Add technical deep-dive with diagrams and benchmark data.
- [ ] **PROD-01**: Include interactive demos and downloadable one-pager.

- Multi-page marketing site architecture
- Automated A/B experimentation
